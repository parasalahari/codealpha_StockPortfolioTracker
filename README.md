# 📈 Stock Portfolio Tracker

## 📌 Project Overview

The **Stock Portfolio Tracker** is a simple Python application developed as part of the **CodeAlpha Python Programming Internship**.

This program allows users to enter stock names and quantities, calculates the total investment value based on predefined stock prices, and optionally saves the result to a text file.

---

## 🚀 Features

* Accepts stock names and quantities from the user.
* Uses a Python dictionary to store stock prices.
* Calculates the total portfolio value.
* Displays the investment value for each stock.
* Saves the total portfolio value to a `portfolio.txt` file.

---
